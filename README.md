#Igraph CMD Layout#

##Installation

1. Run `./configure.sh`
2. Run `./build.sh`

This should do the job.

use the binary `igraph-cmd <Input Edge list FIle> <Output position JSON file>`

##Node JS Bindings##
Has node js bindings to use it easily in a node project.

node module located at `thirdPartyBindings/nodejs`
